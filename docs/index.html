⚡ Getting Started:

  1. Development Server: Click "Start Development Server" or run npm run dev
  2. Production Build: Run npm run build && npm start
  3. Website URL: http://localhost:3000
  4. Stop Server: Press Ctrl+C in the terminal

📚 Installation (First Time Only):

  1. Navigate to the docs folder: cd docs
  2. Install dependencies: npm install
  3. Start development server: npm run dev
  4. Open browser: http://localhost:3000
